# sqlite3_utils

A lightweight Rust wrapper for SQLite3 providing structural CRUD operations.

## Features
- **Root Operations**: Open and delete SQLite databases easily (supports Memory & File).
- **Table Controls**: Create, drop, and modify tables with structured `FieldDescription`.
- **Data Controls**: Insert, update, delete, and query rows mapping to standard Rust Vectors.

## Running Tests
Run standard unit tests:
```bash
cargo test
```
